WDTT — это приложение для создания защищенного туннеля через TURN-серверы VK. Оно инкапсулирует WireGuard-трафик в DTLS-соединения, маскируя его под обычный видеозвонок или медиа-трафик.

### Что видит DPI

```diff
- Без WDTT: Клиент ────────── [ VPN Трафик ] ────────── VPS (185.22.x.x)  ← ❌ БЛОКИРУЕТСЯ

+ С WDTT:   Клиент ── [ DTLS Звонок ] ── 155.212.x.x (VK TURN) ──┄┄[ Скрыто + DTLS ]┄┄── VPS (IP скрыт)
                                         👆                                     👆
                              Виден только этот IP                     Невидим для анализа
```

### 1. VPS

Для работы необходим VPS с любым IP-адресом. 
- 🌐 **IP или Домен** — поддерживается работа как с прямыми IP, так и через доменные имена (FQDN).
- ⚙️ **SSH-Порт** — возможность смены стандартного порта (22) в настройках деплоя.
- 👤 **Логин** — обычно `root`.
- 🔑 **Пароль** — из личного кабинета хостинга.

### 2. Хеш VK-звонка

Вы можете установить от **1 до 3** активных хэшей для повышения пропускной способности и стабильности.

```
1.  Откройте VK → создайте группу (или используйте существующую)
2.  Начните звонок в группе
3.  Скопируйте ссылку — она выглядит как vk.com/call/join/xxxxxxxxxxx
4.  Ваш хеш — код после последнего слэша (xxxxxxxxxxx)
```

> [!IMPORTANT]
> При выходе из звонка нажмите **«Просто завершить»**, а не «Завершить для всех» — иначе звонок закроется и хеш перестанет работать.

### 3. Установка сервера (из приложения)

```
1.  Откройте вкладку «Deploy»
2.  Введите IP/Домен, логин, пароль и SSH-порт VPS
3.  Задайте пароль туннеля в «Секретах»
4.  Нажмите «Установить»
```

### 4. Подключение

```
1.  Введите хеш (до 3-х штук через запятую или автоматически)
2.  Введите пароль туннеля
3.  Нажмите «Подключить»
```
